A sleek, responsive e-commerce storefront landing page designed for a boutique beauty brand. This project showcases a modern UI with a focus on product presentation and user experience.

### ✨ Features
Responsive Grid: Fully fluid layout using Bootstrap 5's grid system, optimized for mobile, tablet, and desktop.

Interactive UI: * Zoom Effect: Smooth scale transformation on product images upon hover.

Elevation Cards: Subtle box-shadow transitions to highlight selected products.

Clean Aesthetic: Minimalist design with a focus on high-quality imagery and typography.

Newsletter Integration: A stylized footer with a "Join the Glow List" subscription form.

### 🛠️ Tech Stack
HTML5: Semantic structure.

CSS3: Custom styling for card animations and layout refinements.

Bootstrap 5: Frontend framework for responsive design and rapid UI development.

Google Fonts/Web Fonts: Clean sans-serif typography.
